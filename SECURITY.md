# Security Policy

## Supported Versions

We will provide security fixes for the latest version of Flow and encourage Flow users through auto-updates to use the latest version of the app.

## Reporting a Vulnerability

Please file security vulnerabilities by using [report a vulnerability](https://github.com/dyad-sh/dyad/security/advisories/new). Please do not file security vulnerabilities as a regular issue as the information could be used to exploit Flow users.
